# Souzou

Souzou aims to be a life management tool. That means that you can use it to manage your life, your work, your projects, your tasks, your notes, your ideas, your everything.

# Build

Install the dependencies:
```bash
npm install
```

Build the Vite app:
```bash
npm run build
```

Create the Electron executable:
```bash
npm run forge:make
```

Run the Electron executable:
```bash
# On Windows
./out/production/Souzou-win32-x64/Souzou.exe
```
